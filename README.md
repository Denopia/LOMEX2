### Install

1) Download the project directory.

2) Go to the main directory "SAKE".

3) To install modified BFCounter, run:

```
cd BFCounterForStrobemers
make
cd ..
```
4) To install SPOA, run:
```
cd SAKEandSPOA/spoa
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
make
cd ../../..


```
5) To install SAKE, run:
```
cd SAKEandSPOA/sake
make sake
cd ../..
```
